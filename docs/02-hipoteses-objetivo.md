# 2. Hipóteses e Objetivo do Projeto

## Hipóteses de Negócio

Durante a análise do cenário pós-fusão, foram levantadas hipóteses que direcionaram a construção da solução de dados:

### Hipótese 1 — Infraestruturas distintas dificultam a integração
**Problema:**  
As empresas operavam em ambientes tecnológicos diferentes (on-premise e cloud), impossibilitando a consolidação direta dos dados.

**Hipótese:**  
A adoção de uma arquitetura híbrida permitiria integrar os dados de forma segura, escalável e performática.

---

### Hipótese 2 — Bases isoladas geram inconsistência de indicadores
**Problema:**  
As bases de dados não se comunicavam, resultando em relatórios divergentes e retrabalho.

**Hipótese:**  
A criação de pipelines de dados padronizados permitiria centralizar e validar as informações, garantindo consistência analítica.

---

### Hipótese 3 — Falta de padronização aumenta a complexidade operacional
**Problema:**  
A ausência de modelos de dados comuns dificultava análises consolidadas.

**Hipótese:**  
A definição de um modelo de dados unificado reduziria a complexidade e facilitaria a geração de KPIs corporativos.

---

### Hipótese 4 — Relatórios manuais comprometem a tomada de decisão
**Problema:**  
A geração manual de relatórios aumentava o tempo de resposta às demandas do negócio.

**Hipótese:**  
A automação do processo de integração e disponibilização dos dados tornaria a tomada de decisão mais ágil e confiável.

---

## Objetivo do Projeto (SMART)

**Objetivo Geral:**  
Integrar os dados de vendas das duas empresas fusionadas em uma base centralizada, padronizada e validada, possibilitando a geração automatizada de relatórios e dashboards unificados para as áreas comercial e executiva.

### Critérios SMART

- **S – Específico:**  
Integração dos dados de vendas em uma base única.

- **M – Mensurável:**  
Redução de divergências nos indicadores e automatização dos relatórios.

- **A – Alcançável:**  
Escopo restrito aos dados de vendas, utilizando tecnologias disponíveis.

- **R – Relevante:**  
Suporte direto à tomada de decisão estratégica pós-fusão.

- **T – Temporal:**  
Execução dentro do prazo definido para o projeto aplicado.
